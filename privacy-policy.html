Politique de confidentialité

Dernière mise à jour : Janvier 2025

1. Collecte des données

Le Camping du Wignet collecte les données suivantes lors de votre réservation :

  • Données personnelles : nom, prénom, email, téléphone, adresse
  • Données de séjour : dates d'arrivée et départ, nombre de personnes
  • Préférences : type d'hébergement, langues parlées, demandes spéciales
  • Données techniques : adresse IP, cookies, données de navigation

Matériel de camping

2. Utilisation des données

Vos données sont utilisées pour :

  • Traiter et confirmer votre réservation
  • Vous contacter concernant votre séjour
  • Améliorer nos services
  • Respecter nos obligations légales
  • Vous envoyer des informations sur les activités locales (avec votre consentement)

3. Google Analytics et AdSense

Notre site utilise :

  • Google Analytics : pour analyser le trafic et améliorer l'expérience utilisateur
  • Google AdSense : pour afficher des publicités personnalisées
  • Cookies : pour mémoriser vos préférences et personnaliser le contenu

Vous pouvez désactiver les cookies dans les paramètres de votre navigateur.

4. Partage des données

Nous ne vendons jamais vos données personnelles. Nous pouvons les partager avec :

  • Nos partenaires de paiement (sécurisation des transactions)
  • Les autorités légales si requis par la loi
  • Nos prestataires techniques (hébergement, envoi d'emails)

5. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : consulter vos données personnelles
  • Droit de rectification : corriger vos données
  • Droit à l'effacement : supprimer vos données
  • Droit de portabilité : récupérer vos données
  • Droit d'opposition : vous opposer au traitement

Pour exercer ces droits, contactez-nous via le formulaire de réservation.

6. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles pour protéger vos données :

  • Chiffrement SSL/TLS pour toutes les communications
  • Accès restreint aux données personnelles
  • Sauvegardes sécurisées régulières
  • Formation de notre équipe aux bonnes pratiques

7. Conservation des données

Nous conservons vos données :

  • Données de réservation : 3 ans après votre séjour
  • Données de facturation : 10 ans (obligation légale)
  • Cookies : 13 mois maximum
  • Données marketing : jusqu'à votre désinscription
