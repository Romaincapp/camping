# Roadmap - Améliorations suggérées

## Court terme

- **Backend réservations** - Firebase/Netlify Functions, email confirmation
- **Optimisation images** - Compression, WebP, srcset
- **UX calendrier** - Feedback loading, animations sélection, tooltips prix

## Moyen terme

- **Multi-langue** - Traduire tout le site (/fr, /en, /nl, /de)
- **Paiement** - Stripe/PayPal, acompte, remboursements
- **Dashboard admin** - Réservations, prix, stats

## Long terme

- **Framework moderne** - Next.js/Nuxt.js, SSR
- **PWA** - Installation mobile, offline, notifications
- **Système reviews** - Avis post-séjour, modération
